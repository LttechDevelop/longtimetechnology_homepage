# 榮炭科技官網：結構設計與頁面修改說明

> 本文件說明官網整體技術架構，以及各頁面內容的修改方式。  
> 閱讀本文件前，建議先閱讀《官網管理交接文件》，了解環境設定與部署流程。

---

## 一、整體技術架構

### 靜態網站 + 元件式組裝

官網沒有任何後端框架，是純 HTML / CSS / JavaScript 靜態網站。每個頁面（`.html`）直接對應一個 URL，不存在動態路由或模板引擎。

頁面之間共用的 Header 和 Footer 以「動態載入」方式引入：每個 HTML 頁面中有一個 `<div data-include="header"></div>`，`js/layout-loader.js` 在頁面載入時會自動 fetch `header.html` 與 `footer.html`，插入對應位置。

因此，**修改導覽列或 Footer 只需編輯這兩個共用檔案**，所有頁面會同步更新。

### 多語系機制

官網支援繁中 / 簡中 / English / 日文，語言切換不重新載入頁面，純靠 CSS class 控制顯示。

**靜態文字（HTML 內容）**

每段需要翻譯的文字，都用 `<span class="tw">...</span>` 等四個 span 並排，CSS 根據 `<body>` 的 `lang_tw` / `lang_cn` / `lang_en` / `lang_jp` class 決定哪一個可見：

```html
<span class="tw">關於LTT</span>
<span class="cn">关于LTT</span>
<span class="en">About Us</span>
<span class="jp">会社概要</span>
```

**導覽列文字（data-lang-key）**

Header 的選單使用 `data-lang-key` 屬性，翻譯字典集中管理在 `js/language-switcher.js` 最上方的 `headerTranslations` 物件中。若要修改選單文字，在這個物件裡找到對應的 key 修改即可：

```javascript
// js/language-switcher.js
const headerTranslations = {
    'menu_about':    { tw: '關於LTT', cn: '关于LTT', en: 'About Us', jp: '会社概要' },
    'menu_products': { tw: '產品介紹', cn: '产品介绍', en: 'Products', jp: '産品内容' },
    // ...
};
```

**頁籤標題（瀏覽器分頁）**

同樣在 `js/language-switcher.js` 的 `pageTitleTranslations` 物件中，對應每個頁面的四語言 `<title>`：

```javascript
const pageTitleTranslations = {
    about: {
        tw: '關於榮炭｜榮炭科技',
        en: 'About Us｜Long Time Technology',
        // ...
    },
    // ...
};
```

## 二、共用元件

### Header（導覽列）— `header.html`

可修改項目：
- **選單項目的連結與順序**：直接修改 `<li><a href="...">` 的 href 或增刪 `<li>` 項目
- **選單文字翻譯**：在 `js/language-switcher.js` 的 `headerTranslations` 修改
- **ESG 頁面**（目前被 `<!-- -->` 注解掉）：若日後要啟用，移除注解符號即可

```html
<!-- header.html — 目前 ESG 被注解 -->
<!--
<li>
    <a href="/esg/" data-lang-key="esg"><span>ESG永續</span></a>
</li>
-->
```

### Footer（頁尾）— `footer.html`

可修改項目：
- **公司電話**：兩處 `<a href="tel:...">` 連結及其 `<span>` 文字
- **傳真號碼**：FAX 的 `<span>` 文字
- **公司地址**：ADD 的四語言 `<span>` 文字
- **Copyright 年份**：底部 `&copy; 2026` 字串，每年更新
- **Google Maps 連結**：地址的 `href="https://maps.app.goo.gl/..."` 換成新地圖短網址

## 三、各頁面說明

### 3.1 首頁 (`index.html`)

首頁主要有以下可修改區塊：

**Banner 標語**

頁面頂部的全幅 banner，有四語言版本的公司願景標語，直接修改對應 `<div class="slogn tw/en/jp/cn">` 內的文字。

**合作夥伴 / 客戶 Logo 區塊**

Logo 圖片存放於 `dist/index/`，HTML 中以 `<img src="dist/index/XXX.png">` 引用。新增或替換 Logo：上傳圖片到 `dist/index/`，在 HTML 中加入對應的 `<img>` 標籤即可。

**最新消息預覽**

首頁的最新消息區塊由 JavaScript 自動讀取 `dist/news-index.json` 並顯示前幾則，**不需要手動修改 HTML**，更新新聞後會自動反映。

---

### 3.2 關於我們 (`about/index.html`)

**3.2.1 管理團隊成員**

每位成員是一個 `<article class="member">` 區塊，包含照片、姓名（四語言）、學歷、經歷：

```html
<article class="member">
    <div class="member-photo">
        <img src="dist/about/姓名.png" alt="姓名 職稱">
    </div>
    <div class="member-content">
        <div class="member-header">
            <h3 class="member-name">
                <span class="tw">王小明<span class="member-role">總經理</span></span>
                <span class="cn">王小明<span class="member-role">总经理</span></span>
                <span class="en">Xiao-Ming Wang<span class="member-role">CEO</span></span>
                <span class="jp">王 小明<span class="member-role">代表取締役</span></span>
            </h3>
        </div>
        <div class="member-sections">
            <div class="member-section member-section--education">
                <!-- 學歷 ul.member-section-list tw/cn/en/jp -->
            </div>
            <div class="member-section member-section--experience">
                <!-- 經歷 ul.member-section-list tw/cn/en/jp -->
            </div>
        </div>
    </div>
</article>
```

**新增成員**：複製一個完整的 `<article class="member">` 區塊，修改內容，並將照片（建議 PNG，正方形比例）放入 `dist/about/`。

**隱藏成員**：將整個 `<article>` 包在 `<!-- -->` 注解符號內，畫面上就不會顯示（已有舊董事長用此方式隱藏）。

**刪除成員**：直接刪除整個 `<article class="member">` 區塊。

**3.2.2 公司大事記（History Timeline）**

大事記是按年份排列的橫向捲動時間軸，每個年份是一個 `<li>` 項目：

```html
<li>
    <div class="ytime">2026</div>
    <div class="dots"><i></i></div>
    <div class="txt">
        <span class="tw">（繁中大事記文字）</span>
        <span class="cn">（简中大事记文字）</span>
        <span class="en">（English milestone text）</span>
        <span class="jp">（日本語マイルストーン）</span>
    </div>
</li>
```

**新增年份**：在 `<ul class="item_menu_list slides yearlist">` 內，依照年份由新到舊的順序，插入新的 `<li>` 項目。

目前已有的年份：2025、2024、2022、2021、2018、2017、2015、2013、2011、2010、2009。

### 3.3 產品介紹 (`products/index.html`)

產品頁為靜態 HTML，直接編輯各產品區塊的文字與圖片即可。

產品圖片存放於 `dist/products/`。

`dist/products/榮炭產品型錄.csv` 是產品規格表的資料來源，若有更新，替換此 CSV 檔案。

### 3.4 投資人專區 (`investors/index.html`)

此頁有四個主要可維護區塊：

**3.4.1 財務報表（`#financial-reports`）**

財務報表以 HTML 表格呈現，PDF 檔存放於 `dist/investors/downloads/`。

**新增一年的財務報表：**

① 將 PDF 上傳至 `dist/investors/downloads/`，依照命名規則：

```
financial-report-YYYY-individual.pdf   ← 年度個體財務報表
financial-report-YYYY-q2.pdf           ← 上半年合併季報
financial-report-YYYY-q4.pdf           ← 全年合併季報
```

② 在 `investors/index.html` 的 `<tbody>` 最上方（最新年份放最前）新增一列：

```html
<tr>
    <th scope="row">2026</th>
    <td>
        <a href="dist/investors/downloads/financial-report-2026-individual.pdf" download>
            <span class="tw">下載</span>
            <span class="cn">下载</span>
            <span class="en">Download</span>
            <span class="jp">ダウンロード</span>
        </a>
    </td>
    <td>
        <div class="quarter-links">
            <a href="dist/investors/downloads/financial-report-2026-q2.pdf" download>Q2</a>
            <a href="dist/investors/downloads/financial-report-2026-q4.pdf" download>Q4</a>
        </div>
    </td>
</tr>
```

若某份報表尚未出爐，可先省略該 `<a>` 連結，待 PDF 備妥後再補上。

**年報（Annual Report）**

年報 PDF 命名沿用民國年格式（例如 `113年度年報.pdf`），存放位置相同，在 `#document-downloads` 區段更新下載連結。

**3.4.2 公司規章下載（`#document-downloads`）**

格式與財務報表相同，直接在對應表格列中更新 `href` 指向新 PDF 即可。

**3.4.3 董事會成員（`#board`）**

每位董事是一個 `<div class="board-card">`：

```html
<div class="board-card">
    <div class="board-role">
        <span class="tw">董事長</span>
        <span class="cn">董事长</span>
        <span class="en">Chairman</span>
        <span class="jp">会長</span>
    </div>
    <div class="board-name">
        <span class="tw">林忠正</span>
        <span class="cn">林忠正</span>
        <span class="en">Chong-zheng Lin</span>
        <span class="jp">林 忠正</span>
    </div>
    <div class="board-org">
        <span class="tw">英特盛科技股份有限公司</span>
        <span class="cn">英特盛科技股份有限公司</span>
        <span class="en">Intense Technology Co., Ltd.</span>
        <span class="jp">インテンステクノロジー株式会社</span>
    </div>
</div>
```

修改、新增、刪除董事：直接在 `.board-grid` 內操作對應的 `.board-card` 區塊。

### 3.5 聯絡我們 (`contact/index.html`)

**聯絡表單**

表單透過隱藏 iframe 提交至 **Google 試算表（Google Forms）**。若 Google 試算表連結需要更換，在 `contact/index.html` 找到 `<form id="contact"` 的 `action` 屬性，替換成新的 Google Forms 提交 URL。

**Google Maps 嵌入地圖**

地圖是 `<iframe src="https://www.google.com/maps/embed?pb=...">` 嵌入。若公司地址搬遷，到 Google Maps 取得新地點的「嵌入地圖」iframe 代碼，替換整個 `<iframe>` 標籤即可。

## 四、SEO 設定修改

每個頁面的 SEO 資訊都寫在各自 HTML 的 `<head>` 區塊，主要有以下欄位：

```html
<meta name="description" content="頁面描述，150字以內">
<meta property="og:title" content="頁面標題｜榮炭科技">
<meta property="og:description" content="頁面描述">
<meta property="og:url" content="https://www.lttech.com.tw/XXX/">
<meta property="og:image" content="https://www.lttech.com.tw/dist/Website_Metadata_Images/ogimg.jpg">
<link rel="canonical" href="https://www.lttech.com.tw/XXX/">
```

**OG 預覽圖**（Facebook / LINE 分享時顯示的縮圖）存放於：

```
dist/Website_Metadata_Images/ogimg.jpg
```

若要更換縮圖，替換這個檔案即可（建議尺寸 1200×630 px）。

**網站 favicon** 存放於：

```
dist/Website_Metadata_Images/16x16.ico
```

## 六、JavaScript 檔案功能索引

| 檔案 | 功能 |
|------|------|
| `js/layout-loader.js` | 動態載入 header.html / footer.html |
| `js/language-switcher.js` | 語言切換邏輯、選單文字字典、頁籤標題字典 |
| `js/layout.js` | 導覽列互動（展開/收合、active state） |
| `js/function.js` | 共用工具函式 |
| `js/main.js` | 各頁初始化入口 |
| `js/investors-page.js` | 投資人頁的股價/營收圖表（讀取 CSV） |
| `js/news-index.js` | 新聞列表頁（讀取 news-index.json） |
| `js/news-detail-page.js` | 新聞內頁（讀取單則 .md） |
| `js/contact-form.js` | 聯絡表單提交邏輯 |
| `js/about.js` | 關於我們頁的 slider 與互動 |
| `js/security-headers.js` | 前端安全性設定 |

一般內容維護不需要修改任何 JS 檔案。除非有功能性需求，否則避免動到這些檔案。
